# Generator Questów do Gier

Projekt generatora questów do gier, umożliwiający tworzenie różnorodnych zadań i przygód dla postaci gracza.

## Lista pól rozwijanych:

1. **Typ gry:**
   - [ ] Akcja
   - [ ] RPG
   - [ ] Strategiczna
   - [ ] Przygodowa

2. **Typ questa:**
   - [ ] Zabójstwo
   - [ ] Zbieranie
   - [ ] Eksploracja
   - [ ] Interakcja NPC

3. **Ilość części questa:**
   - [ ] Jedna
   - [ ] Wieloetapowy

4. **Trudność:**
   - [ ] Łatwy
   - [ ] Średni
   - [ ] Trudny

5. **Zaangażowane przedmioty:**
   - [ ] Tak
   - [ ] Nie

6. **Zaangażowane NPC:**
   - [ ] Tak
   - [ ] Nie
